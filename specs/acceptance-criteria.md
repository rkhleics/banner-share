# Acceptance Criteria

## Upload
- Non-technical user can upload a ZIP successfully
- Clear error messages for invalid ZIPs
- Progress feedback during upload

## Review
- One URL opens all banners on one page
- Restart works for every banner
- Pause/Play works where CreateJS allows
- Failures degrade gracefully

## Technical
- Same-origin iframe access confirmed
- No CORS errors
- Assets load correctly from proxy routes
